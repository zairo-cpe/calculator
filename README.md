# 📱 Simple Web Calculator

A sleek, lightweight web-based calculator featuring a modern, comfortable glassmorphic UI. Built purely with semantic HTML, custom CSS styling, and vanilla JavaScript logic.

🔗 **[Live Demo](https://zairo-cpe.github.io/calculator/)**

## 🚀 Features & Functionality

This project handles basic arithmetic operations entirely client-side using JavaScript variables and logical operators. 

### Supported Operations:
* ➕ **Addition** (`+`)
* ➖ **Subtraction** (`-`)
* ✖️ **Multiplication** (`*`)
* ➗ **Division** (`/`)

---

## 🛠️ Tech Stack

* **HTML5:** Structured the calculator layout and interactive buttons.
* **CSS3:** Styled the smooth gradient background and the semi-transparent frosted glass (glassmorphism) containers.
* **JavaScript (ES6):** Handled input capturing, state management, and arithmetic evaluations.
